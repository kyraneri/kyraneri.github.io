Strings

A String contains text using letters, numbers, symbols, or puncuration.

A string can use single quotes or double quotes.

While, Javascript can use both single & double quotes, a project should use only one type.

  • Example:
  • let text = "This is a string";
Numbers

JavaScriptJavaScriptJavaScriptJavaScriptNumbers can include decimals.

The Number() method will convert variables to numbers. For example: Number(10) would return 10.

  • Example:
  • let x = 123;
  • let x = 1.23;
Booleans

There are only two possible values for a boolean. True or False. Booleans are often used for conditional testing.

A boolean can be an object when defined with the new keyword.

Undefined

When a variable has no value, the value and type are undefined.

  • Example:
  • let text;

A variable can also be emptied by setting te value to undefined

Variables

A variable stores data/values.
There are 4 ways to declare a variable in JavaScript. Using var, let, const, or nothing. In order for the code to function on older browser, var must be used. Since let and const were added to JavaScript in 2015.

  • Example:
  • var x = 1;
  • let x = 2;
  • const x = 3;
  • x = 4;
Arrays

Arrays are Object Datatypes.

Arrays use square brackets [], separate items using commas, and the first item begins with 0.
